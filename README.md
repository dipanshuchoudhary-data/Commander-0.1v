Overview

Commander 0.1v is the first experimental prototype of a local voice command system.
This version is not polished, not optimized, and is intended only as a proof of concept.

The goal of this release is to validate:

Microphone input handling

Local speech-to-text (offline)

Basic command execution

Future versions will focus on reliability, usability, and accuracy.
